# TruckersFifthWheel

This is the front-end and back-end code to a project I worked on during the summer of 2019. 

The project used to be deployed for a few years but is now offline. I closed it for security reasons but I will soon relaunch it! I have posted a few screenshots to show how the website used to look like.

## Built Features

These are the features that were built for this project

* Creating new accounts
* Editing account information
* Authentication
* Creating and Viewing social media posts
* Commenting on posts
* Liking posts
* Creating and viewing jobs
* Chat
